NEW-002 인보이스 AED(디르함) 통화 발행 ▼

기능 설명

아랍권(UAE) 거래 시 인보이스를 USD 대신 AED(디르함)로 발행합니다. 고정 환율 1 USD = 3.6725 AED로 자동 변환됩니다.

사용 방법

  1. 바이어 관리 → 주문 상세 → 서류관리 섹션으로 이동
  2. "AED(디르함) 발행" 체크박스를 선택
  3. 인보이스 발행 버튼 클릭
  4. 생성된 PDF에 AED 금액 + 환율 정보가 표시됨

변환 예시

USD 금액AED 금액계산
$50,000.00183,625.00 AED50,000 x 3.6725
$1,400.005,141.50 AED1,400 x 3.6725
$500.001,836.25 AED500 x 3.6725
IBK 은행계좌, 선박명 제거와 동시에 사용 가능합니다. 3개 체크박스를 모두 선택할 수 있습니다.

NEW-007 인쿼리 자동 파싱 - 개요 및 사용법 ▼

기능 설명

인쿼리(견적 요청) 텍스트를 붙여넣으면 선박명, 항구, 유종, 수량, 가격, 결제조건 등을 자동으로 추출하여 바이어/셀러 주문을 생성합니다. 수작업 입력 없이 복사+붙여넣기만으로 주문이 만들어집니다.

NEW-013 ETA 필드 + 3단계 DDD 자동 계산 ▼

기능 설명

주문 상세에 ETA(선박 도착 예정일) 필드가 추가되었습니다. ETA, 공급예정일, 공급확정일 중 하나가 변경되면 지급기한일(PayDueDate)이 자동으로 재계산됩니다.

사용 방법

  1. 바이어 관리 → 주문 클릭 → 상세 팝업 열기
  2. ETA 날짜 입력 (공급요청일과 최종완료일 사이에 위치)
  3. 날짜 선택 즉시 → 지급기한일이 자동 갱신됨
  4. 저장하면 ETA 값이 DB에 저장됨

지급기한일 계산 우선순위

아래 순서로 기준일을 선택하여 DDD를 계산합니다. 상위 값이 있으면 하위 값은 무시됩니다.

우선순위기준일설명
1순위공급확정일 (SupplyCompleteDate)실제 공급이 완료된 날짜
2순위공급예정일 (EstSupplyStartDate)공급 예정 시작일
3순위ETA (선박 도착 예정일)선박이 항구에 도착하는 예정일
4순위공급요청일 (ReqSupplyStartDate)바이어가 요청한 공급일
